# STEA Project Tests

This directory contains various test scripts used during the development of the STEA project.

## Useful Tests

| File | Description |
|------|-------------|
| `api-fetch-test.js` | Generic script to test API endpoints. |
| `necta-crawler.js` | Checks fallback NECTA result URLs on maktaba.tetea.org. |
| `site-e2e-test.mjs` | Puppeteer script to verify the site is running and rendering content. |

## Mocks
The `/tests/mocks` directory contains mocked Node.js modules used for compatibility in certain environments.

## Archive
The `/tests/archive` directory contains old or redundant test scripts that are no longer actively used but kept for reference.
